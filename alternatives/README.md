# Alternatives (non–Chrome-extension)

> English · [中文](./README.zh-CN.md)

Fallback injectors when you cannot install the extension. Same ideas as `extension/unlock.js`; keep them in sync yourself.

| File | Usage |
|------|--------|
| `devtools-unlock.console.js` | Paste in DevTools Console, or inject via Chrome Overrides at `document_start` |
| `devtools-unlock.userscript.js` | Tampermonkey / Violentmonkey userscript |

The extension is still recommended for daily use: stable injection timing and a one-click per-site toggle.
